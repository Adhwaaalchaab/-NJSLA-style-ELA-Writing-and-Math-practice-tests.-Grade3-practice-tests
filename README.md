# NJSLA Grade 3 Practice Tests

Interactive Grade 3 NJSLA-style practice tests for students preparing for the New Jersey Student Learning Assessments.

## Included Practice Tests

- ELA Writing Practice
- Mathematics Practice

## Features

- Timed test sessions
- Multiple practice forms
- Multiple-choice questions
- Multi-select questions
- Drag-and-drop style questions
- Multi-step math problems
- Writing response practice
- Automatic scoring and feedback

## Important Disclaimer

These materials are original practice resources. They are not official NJSLA exams and do not copy secure state test items.

The purpose of this repository is to provide students with practice that is similar in structure and skill coverage to public NJSLA-style assessments.

## How to Use

Open the website and select a practice test:

- ELA Writing
- Math

Students can complete the test in the browser and review their score after submission.

## Recommended Use

This resource is best used for:

- Grade 3 test preparation
- Classroom practice
- Home practice
- Tutoring
- Familiarity with online test formats
